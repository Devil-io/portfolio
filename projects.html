Hamburger Menu

This is a webpage that includes a hamburger menu for navigation and a grid of images.
The hamburger menu is a popular UI pattern used in responsive web design to toggle a navigation menu on and off for smaller screen sizes. When clicked, it reveals a set of links for navigation. In this code, the hamburger menu is implemented using a checkbox input element and spans styled to resemble the three lines typically associated with the hamburger icon.

Digital Clock

This webpage displaying a real-time clock with an option to toggle between light mode and dark mode.
The clock itself is displayed in a div element with the id "clock". JavaScript code is used to update the clock's display every second using the setInterval function.
Below the clock, there's a mode toggle feature implemented using a checkbox input element. When toggled, it changes the background color of the page to simulate switching between light mode and dark mode. The text of the mode toggle button changes accordingly to reflect the current mode.
